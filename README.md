# South Brew — Café Website

A professional, fully responsive website built for **South Brew**, a local neighborhood café and roastery. This project was built to demonstrate how a simple, well-designed website can help a small business look more professional, build customer trust, and grow its online presence.

**Live demo:** https://south-brew-71b9e3.netlify.app/

## Why This Project

Many local cafés, salons, and small shops either have no website or an outdated one, which makes it harder for new customers to find them, trust them, and know their menu, hours, or location before visiting. This project solves that with a clean, modern, single-page site built specifically around South Brew's brand and offerings.

## Goals

- **Look more professional** — custom design instead of a generic template
- **Attract customers** — clear menu, hours, and location visible to anyone searching online
- **Increase trust** — customer reviews, real business details, and a polished first impression
- **Grow online** — a shareable link for social media, Google Business, and word of mouth

## Features

- Hero section highlighting the café's daily featured brew
- Interactive tabbed menu (Coffee / Filter Brews / Food)
- About section telling the café's story and stats
- Customer reviews / testimonials section
- Visit section with embedded map, hours, phone number, and a call-to-reserve button
- Fully responsive design (mobile, tablet, desktop)
- Accessible design with visible keyboard focus states

## Tech Stack

- HTML5
- CSS3 (custom design system, no frameworks)
- Vanilla JavaScript (for menu tabs and mobile navigation)
- Google Fonts (Fraunces + Work Sans)

## Running Locally

Clone the repo and open the file directly in a browser:

```bash
git clone https://github.com/<your-username>/south-brew-cafe.git
cd south-brew-cafe
open index.html
```

No build step or dependencies required — it's a static site.

## Deployment

This site is deployed on **Netlify** and can also be hosted via **GitHub Pages**:

1. Go to repo **Settings → Pages**
2. Source: Deploy from a branch → `main` / root
3. Live at `https://<your-username>.github.io/south-brew-cafe/`
